# AI Skill 技能发现助手

帮助中文用户在 88,000+ 个 Agent 技能中，快速找到最合适的那一个。

## 安装

```bash
npx skills add 1-SKILL/ai-skill -g -y
```

## 功能

- 🔍 **智能搜索** — 用中文描述你想做的事，自动匹配最合适的技能
- 🛡️ **安全检查** — 7 项质量验证，确保推荐的技能可靠安全
- 🇨🇳 **中文优先** — 专为中文用户设计，覆盖小红书、公众号、飞书等场景
- 📦 **一键安装** — 找到就能装，装完就能用
- 🔄 **远程更新** — 精选推荐实时更新，无需重装

## 使用

安装后，用自然语言告诉 AI 你需要什么能力：

- "有没有能帮我写小红书的技能？"
- "找个做 PPT 的 skill"
- "AI 能不能帮我做 SEO？"
- "推荐几个好用的技能"
